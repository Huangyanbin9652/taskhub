# TaskHub - 有趣任务发布平台

一个轻量级的有趣任务发布与接单平台，支持发布任务、接单挑战、评论互动、积分排行榜。

## 技术栈

- **后端**: Node.js + Express
- **数据库**: SQLite (better-sqlite3)
- **前端**: 原生 HTML/CSS/JS，移动端优先响应式设计

## 本地运行

```bash
npm install
npm start
```

打开 http://localhost:3000 即可访问。

## 体验账号

## 功能

- 🏠 任务广场（分类筛选、搜索）
- ➕ 发布任务（自定义分类、奖励、难度）
- 🤝 接单挑战
- 💬 评论互动
- 🏆 积分排行榜
- 👤 个人中心（自选头像、编辑简介）
- 📱 移动端适配（响应式 + 触摸优化）

## 部署

适配 Render / Railway 等平台，设置环境变量 `PORT` 即可。
